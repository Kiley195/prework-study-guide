HTML

  • The head element contains information abour the webpage.
  • The body element represents the visible content shown to the user.
  • Child elements are elements inside of other elements.
  • Text, image, header, and list elements are used to add content to a page.
  • Meta-related, content division, and semantic elements divide and describe a page.
  • Metadata helps the browser render the page correctly, it is not seen by users.
  • Headers are h1 - h6, the numbers represent the level of heading. These should be used in order and h1 should only be used once per page.
  • Semantic HTML elements: the name of the element indicates role of content as well as providing structure to the content, these should be used where ever possible.
  • When adding an image the 'src' attribute defines it's location or source; the 'alt' attribute contains a text string the describes the image. The 'alt' text is important for users using assistive technologies and to help improve search engine rankings.
  • 'Section' elements define a sectio in a document that contains thematic content.

CSS

  • A margin indicates how much space we want around the outsde of an element.
  • A padding indicates how much space we want around the content iside an element.
  • Inline CSS styles apply to a specific HTML tag to style a particular page element only.
  • Internal style sheets are located in the head element, the rules only apply to that page.
  • External style sheets are separate files linked in the HTML head element.
  • A CSS rule is built from a selector followed by declarations. The selector defines the element or attributes the declarations will apply to.
  • Declarations contain two components: the CS property we want to apply and the value of the property.
  • * can be used to apply a rule to ALL elements visible on a page.
  • Two or more elements that share the same rule can be grouped together using commas.
  • Class attributes allow you to share a CSS rule to any element by assigning the rule to a class attribute with a class selector. The class attribute can then be assigned to any element you wish to be assigned the CSS rule.
  • Using a period before a class designates it as a class selector. Any element assigned to that class will follow the CSS rules applied to the class selector.

Git

  • cd "name of directory": navigate to that directory
  • git status:checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git clone [url]: retrieve an entire repository from a hosted location via URL
  • git add -A: add modifications made in the current working branch to the staging area
  • git commit -m "[descriptive message]": commit staged content
  • git pull origin "branch you want to pull from": pulls changes from base branch to local environment - ensures you have the latest version of the project on your local pc
  • git push origin "branch you want to push to" : push changes made locally to remote GitHub branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript files are executed from top to bottom.
  • When we create a function, we are defining a block of code that performs a specific task and giving it a unique name.
  • To execute a function, it must be called in the code.
  • An array allows you to store a collection of items under a single variable name, and can contain a mix of data types.
  • Arrays are zero-indexed, meaning the first element in an array is at index 0.
  • Strings are used to store text.
  • A boolean is a true or false value, most often used in conditional logic.
  • Variables can be declared using let, const, or var. Let allows you to reassign the variable. Const declares a variable whose values are never intended to hange, it cannot be reassigned.
  • Console.log() outputs the values of what we put inside the parentheses to the console.
  • If statements will only execute the code if the statement meets a condition. Else if an be used to introduce more than one condition.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single block of code to executed repeatedly.
  • A for loop contains three statements: The first determinse the starting point of the loop (i.e. let x = 0). The second is the condition (i.e. as long as the variable is less than the length of the array x < array.length).
